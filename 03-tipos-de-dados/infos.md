## Tipos de Dados

### Inteiros

Os **números inteiros** podem possuir diferentes classificações como: *int8*, *int16*, *int32*, *int64* e, puramente, *int*.

Suas únicas diferenças são suas capacidades de tamanho, sendo: 
*int8* = -128 a 127
*int16* = -32768 a 32767
*int32* = -2147483648 a 2147483647
*int64* = -9223372036854775808 a 9223372036854775807
*int* = -9223372036854775808 a 9223372036854775807 (**int64**)

Ainda em **números inteiros**, as classificações **unsygned int**: *uint8*, *uint16*, *uint32*, *uint64* e, puramente, *uint*. Sendo: 
*uint8* = 0 a 255
*uint16* = 0 a 65535
*uint32* = 0 a 4294967295
*uint64* = 0 a 18446744073709551615
*uint* = 0 a 18446744073709551615 (**uint64**)


### Reais

Possue apenas dois tipos: *float32*, *float64*.
